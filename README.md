# petaura-landing-page
Website oficial da PetAura: Pet Shop Premium. Landing page de alta conversão desenvolvida com HTML5, Tailwind CSS e foco em agendamentos via WhatsApp. Design responsivo e otimizado para a melhor experiência do usuário e de seus pets.
🐾 PetAura - Pet Shop Premium 

​Onde o cuidado e o carinho se encontram para mimar quem você ama. Este repositório contém o código-fonte da landing page oficial da PetAura, um site desenvolvido com foco em conversão, agendamentos rápidos e uma experiência visual acolhedora para donos de pets.

​🐶 Sobre a PetAura 

​A PetAura é mais do que um pet shop; é um refúgio de bem-estar animal. O site reflete essa missão, destacando serviços de Estética Canina, Saúde Preventiva e uma Boutique exclusiva, sempre priorizando a transparência e o amor pelos animais.

​🚀 Tecnologias e Ferramentas 

​Desenvolvido com tecnologias modernas para garantir performance e adaptabilidade:

​HTML5 & CSS3: Base sólida e semântica. ​Tailwind CSS: Estilização responsiva com uma paleta de cores vibrante (Teal e Orange). ​Lucide Icons: Ícones modernos para navegação intuitiva. ​JavaScript: Lógica para menu hambúrguer mobile e interações de FAQ (Acordeão). ​Google Fonts: Utilização da fonte Poppins para uma leitura amigável e moderna. ​✨ Diferenciais do Projeto ​🎯 Foco em Conversão (CTA): Botões estratégicos para agendamento via WhatsApp com mensagens personalizadas. ​📱 Mobile-First: Design totalmente otimizado para dispositivos móveis, garantindo que o cliente agende de qualquer lugar. ​🎁 Seção de Oferta Irresistível: Layout destacado com gatilhos de urgência para cupons de primeira visita. ​📹 Destaque de Diferenciais: Espaço para mencionar monitoramento por câmeras, Taxi Dog e higiene rigorosa. ​🙋 FAQ Dinâmico: Seção de dúvidas frequentes com sistema de abertura e fechamento suave. ​📺 Demonstração
